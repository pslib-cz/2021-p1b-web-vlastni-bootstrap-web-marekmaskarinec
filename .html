klak


Klak is a concatenative programming language. It is compiled to c, but I plan
on adding js support if I ever want to suffer. It is in working-ish state.

umka libs and utils


I wanted to try doing something like cc's -MM, but for umka in umka.
Because I wanted to do it as fast as possible, I decided to use my favorite
strings.split parsing technique. There was one problem: there was no string
library for umka. So I wrote one. Then I wrote another and another and I
ended up with a good collection of not only libraries but utilities.
