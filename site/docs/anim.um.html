anim.um

Module for frame based image animation.

type Anim*

type Anim* = struct {
	anim: []image.Image
	rot: th.fu
	scale: th.fu
	fps: th.uu
	looping: bool

	pos: th.Vf2

	zeroTime: int

	frame: int
}

Anim is able to animate an array of images. As opposed to ent.Ent.animate, it starts from the first frame.
