The tophat engine

A friendly 2d game engine for making games in the umka programming language.

Find out more Downloads

Features

Very simple to use

Tophat doesn't require you to learn an editor, or a giant api. Yet it still remains fast and powerful enough to be used in game development.

Modular design

Tophat is build from modules. Most of them are just abstractions over a few base modules. You can make your own modules or replace the standard ones you don't like.

Open and hackable

Tophat's source code is available under the BSD-3 license. It's only about 6000 lines and is very easy to understand and edit. The source code is available on github.

Built-in UI framework

Tophat features a very simple, but very easy to extend ui framework. You can read an article about it here.
